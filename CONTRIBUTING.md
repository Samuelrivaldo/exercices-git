# 🤝 Guide de Contribution\n\nMerci de s'intéresser à ce projet! Voici comment vous pouvez contribuer.\n\n## 📋 Avant de commencer\n\nLisez le [README.md](README.md) pour comprendre le projet.\n\n## 🚀 Comment Contribuer\n\n### 1. Clonez le projet\n```bash\ngit clone https://github.com/Samuelrivaldo/exercices-git.git\ncd exercices-git\n```\n\n### 2. Créez une branche\n```bash\ngit checkout -b feature/votre-fonctionnalite\n```\n\n### 3. Faites vos modifications\n- Modifiez les fichiers pertinents\n- Testez dans le navigateur\n\n### 4. Commits clairs\n```bash\n# Exemples de bons commits\ngit commit -m \"Ajouter validation du formulaire\"\ngit commit -m \"Corriger affichage sur mobile\"\ngit commit -m \"Améliorer accessibilité ARIA\"\n```\n\n### 5. Push et Pull Request\n```bash\ngit push origin feature/votre-fonctionnalite\n```\n\n## 💡 Idées d'Améliorations\n\n- [ ] Mode sombre\n- [ ] Catégories de tâches\n- [ ] Priorités\n- [ ] Date d'échéance\n- [ ] Filtres avancés\n- [ ] Syncronisation cloud\n- [ ] Notifications\n- [ ] Histogramme de productivité\n- [ ] Support multi-langue\n- [ ] Thèmes personnalisés\n\n## 📝 Critères de Qualité\n\n- ✅ Code lisible et commenté\n- ✅ Testé dans navegateur\n- ✅ Point virgules et indentation correctes\n- ✅ Pas de console.log laissés\n- ✅ Message de commit clair\n\n## 🐛 Signaler un Bug\n\nOuvrz une issue avec:\n- Description détaillée\n- Étapes pour reproduire\n- Navigateur et version\n- Capture d'écran si possible\n\n## 📞 Questions?\n\nOuvrz une discussion ou une issue!\n\nHeureux de coder! 🎉\n"
