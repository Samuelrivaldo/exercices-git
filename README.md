# Exercice Git - Gestionnaire de Tâches

> Un projet éducatif pour apprendre et pratiquer Git, tout en créant une application web fonctionnelle.

## 🎯 Objectif

Ce projet permet d'apprendre les bases de Git (commit, push, pull, branches) tout en développant une application web simple et progressive.

## ✨ Fonctionnalités

- ✅ Interface utilisateur moderne et responsive
- ✅ Gestion de tâches (ajouter, supprimer, cocher)
- ✅ Stockage local des données (localStorage)
- ✅ Design élégant et intuitif
- ✅ Compatible avec tous les navigateurs modernes

## 🚀 Démarrage Rapide

### Prérequis
- Git installé
- Un navigateur web moderne
- Un éditeur de code (VS Code recommandé)

### Installation

```bash
# Cloner le référentiel
git clone https://github.com/Samuelrivaldo/exercices-git.git

# Accéder au dossier du projet
cd exercices-git

# Ouvrir dans le navigateur
# Double-cliquez sur index.html ou utilisez un serveur local
```

## 📚 Exercices Git Suggérés

### Niveau 1 - Débutant
1. Cloner le projet
2. Créer une branche locale : `git checkout -b feature/mon-amelioration`
3. Faire des modifications
4. Faire un commit : `git commit -m "description claire"`

### Niveau 2 - Intermédiaire
1. Créer plusieurs commits pour différentes fonctionnalités
2. Fusionner les branches : `git merge`
3. Gérer les conflits

### Niveau 3 - Avancé
1. Utiliser `rebase` et `cherry-pick`
2. Écrire un bon historique de commits
3. Utiliser les tags de version
