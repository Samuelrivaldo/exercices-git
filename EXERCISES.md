# 📚 Exercices Git\n\nCe document propose une série d'exercices pour maîtriser Git en utilisant ce projet.\n\n## 🟢 Niveau 1 - Débutant\n\n### Exercice 1.1 - Clone et Navigation\n```bash\n# Clone le projet\ngit clone https://github.com/Samuelrivaldo/exercices-git.git\ncd exercices-git\n\n# Affiche l'historique\ngit log --oneline\n\n# Affiche les branches\ngit branch -a\n```\n\n### Exercice 1.2 - Créer et Commiter\n```bash\n# Crée une branche\ngit checkout -b feature/add-comments\n\n# Modifie un fichier (ajoute un commentaire dans main.js)\n# ...\n\n# Vérifie les modifications\ngit status\n\n# Stage le fichier\ngit add main.js\n\n# Commit\ngit commit -m \"Ajouter des commentaires au TaskManager\"\n\n# Affiche le dernier commit\ngit log -1 -p\n```\n\n### Exercice 1.3 - Configuration Git\n```bash\n# Configure votre identité\ngit config --global user.name \"Votre Nom\"\ngit config --global user.email \"votre.email@example.com\"\n\n# Affiche la configuration\ngit config --list\n```\n\n## 🟡 Niveau 2 - Intermédiaire\n\n### Exercice 2.1 - Branches Multiples\n```bash\n# Crée deux branches features différentes\ngit checkout -b feature/dark-mode\necho \"/* Dark mode styles */\" >> style.css\ngit add style.css\ngit commit -m \"Ajouter styles mode sombre\"\n\n# Change de branche\ngit checkout master\n\n# Crée une autre branche\ngit checkout -b feature/export-csv\necho \"// Export CSV\" >> main.js\ngit add main.js\ngit commit -m \"Ajouter fonction d'export CSV\"\n```\n\n### Exercice 2.2 - Fusion (Merge)\n```bash\n# Place-toi sur master\ngit checkout master\n\n# Fusionne une branche\ngit merge feature/dark-mode\n\n# Affiche le graphique\ngit log --graph --oneline --all\n```\n\n### Exercice 2.3 - Gérer les Conflits\n```bash\n# Crée deux branches avec modifications au même endroit\ngit checkout -b conflict-test-1\n# Modifie le même fichier\ngit commit -am \"Modification 1\"\n\ngit checkout master\ngit checkout -b conflict-test-2\n# Modifie le MÊME fichier différemment\ngit commit -am \"Modification 2\"\n\n# Essaie de merger\ngit merge conflict-test-1\n# Résous les conflits dans l'éditeur\n# git add.\n# git commit\n```\n\n## 🔴 Niveau 3 - Avancé\n\n### Exercice 3.1 - Rebase\n```bash\n# Crée une branche de travail\ngit checkout -b feature/rebase-practice\necho \"Modification 1\" > test.txt\ngit commit -am \"Commit 1 dans feature\"\n\n# Sur master\ngit checkout master\necho \"Nouveau commit\" >> README.md\ngit commit -am \"Mise à jour README\"\n\n# Rebase la branche\ngit checkout feature/rebase-practice\ngit rebase master\n```\n\n### Exercice 3.2 - Cherry-pick\n```bash\n# Affiche les commits disponibles\ngit log --oneline master\n\n# Cherry-pick un commit spécifique\ngit cherry-pick abc1234\n```\n\n### Exercice 3.3 - Interactive Rebase\n```bash\n# Affiche les 3 derniers commits\ngit rebase -i HEAD~3\n\n# Là tu peux:\n# - reorder (réordonner)\n# - squash (fusionner)\n# - reword (renommer)\n# - drop (supprimer)\n```\n\n### Exercice 3.4 - Stashing\n```bash\n# Modifie un fichier sans commiter\necho \"Travail en cours\" >> main.js\n\n# Sauvegarde temporairement\ngit stash\n\n# Fais autre chose\ngit checkout autre-branche\n\n# Reviens et récupère tes modifications\ngit checkout feature\ngit stash pop\n```\n\n## 🎓 Exercices Création\n\n### Challenge 1 - Clean History\nCrée une branche avec 3 commits, puis utilise rebase interactive pour les fusionner en un seul commit bien nommé.\n\n### Challenge 2 - Partial Commit\nModifie 3 fichiers, mais commit-les séparément avec `git add -p` (partial staging).\n\n### Challenge 3 - Branch Cleanup\nCrée 5 branches, puis supprime-les correctement avec git branch -d et -D.\n\n### Challenge 4 - Tag Versions\nCrée des tags pour simuler des versions:\n```bash\ngit tag v1.0.0\ngit tag v1.1.0\ngit tag -l\n```\n\n## 📖 Ressources Utiles\n\n- [Git Documentation](https://git-scm.com/doc)\n- [Atlassian Git Tutorials](https://www.atlassian.com/git/tutorials)\n- [Pro Git Book](https://git-scm.com/book/en/v2)\n- [GitHub Learning Lab](https://lab.github.com/)\n\n## 💬 Besoin d'Aide?\n\nUt des questions?
