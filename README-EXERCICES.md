# 📚 Exercices Git & GitHub pour Débutants

**Bienvenue!** Ce projet contient une série d'exercices pratiques pour apprendre Git et GitHub de zéro.

## ⭐ Qu'est-ce que Git?

Git est un **gestionnaire de versions** qui vous permet de:
- 📝 Suivre chaque changement dans votre code
- 🔄 Revenir à une version antérieure si nécessaire
- 👥 Collaborer avec d'autres développeurs
- 🌳 Créer des branches pour tester des idées

## 🎯 Structure des Exercices

```
📘 GUIDE_COMPLET.md         ← Lisez ça en premier!
📋 AIDE-MEMOIRE.md          ← Pense-bête des commandes

📁 EXERCICE-1-basics/       - Les commandes de base (1h)
📁 EXERCICE-2-staging-area/ - Comprendre l'index (1h)
📁 EXERCICE-3-commit/       - Bien committer (30min)
📁 EXERCICE-4-branches/     - Créer des branches (1h)
📁 EXERCICE-5-merge/        - Fusionner du code (1.5h)
📁 EXERCICE-6-github-sync/  - Synchroniser avec GitHub (1h)
📁 EXERCICE-7-pull-request/ - Pull Requests (1h)
```

## 💡 Conseils pour bien apprendre

✅ **À FAIRE:**
- Tapez TOUTES les commandes (ne les copiez pas!)
- Utilisez `git status` après chaque commande
- Créez vos propres fichiers pour expérimenter
- Relisez les concepts après chaque exercice
- Posez-vous: "Pourquoi je dois faire ça?"

❌ **À ÉVITER:**
- Copier-coller les commandes
- Sauter des étapes
- Croire qu'on peut comprendre sans pratiquer
- Avoir peur de faire des erreurs (c'est normal!)

## 🆘 Besoin d'aide?

### Pour chaque exercice:
1. Lisez le `README.md` complètement
2. Suivez les étapes une par une
3. Si ça ne marche pas, consultez `SOLUTION.md`

→ Installez Git: https://git-scm.com
